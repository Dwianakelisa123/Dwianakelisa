Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lai, yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf,jarak antar elemen, dan masih banyak lagi.
Css terdiri dari atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML melainkan sebagai pelengkap agar tampilan HTML menjadi lebih indah dan dinamis. berikut ini adalah contoh dokumen CSS.
